# 路由器常见问题知识库

## WAN 口无连接

现象：Wi-Fi 可以连接，但手机或电脑无法访问互联网；路由器指示灯橙色、红色或持续闪烁。

低风险排查：

1. 检查光猫或上级网络设备是否在线。
2. 确认网线插在路由器 WAN 口，不是 LAN 口。
3. 更换网线测试。
4. 先重启光猫，等待光猫稳定后再重启路由器。
5. 进入管理后台检查上网方式：DHCP、PPPoE、静态 IP。

停止条件：设备冒烟、焦味、进水、适配器异常发热；不要拆开路由器或电源适配器。

## Wi-Fi 能连但速度慢

优先检查：距离、墙体遮挡、2.4G/5G 频段、信道拥堵、终端数量、后台下载任务。

建议步骤：靠近路由器测试；重启路由器；切换 5G Wi-Fi；检查是否有陌生设备占用；更新固件。
